Sort the numbers in an array. But the position of zeros should not be changed.

Input: {% if interpreter.slug == "js-node" %}An Array{% else %}A List{% endif %}.

Output: An {% if interpreter.slug == "js-node" %}Array{% else %}Iterable (tuple, list, iterator ...){% endif %}.

Example:

{% if interpreter.slug == "js-node" %} 
exceptZero([5, 3, 0, 0, 4, 1, 4, 0, 7]) == [1, 3, 0, 0, 4, 4, 5, 0, 7]
exceptZero([0, 2, 3, 1, 0, 4, 5]) == [0, 1, 2, 3, 0, 4, 5]
