Ok, what should I start from?

Let's create const cleaned (you are not going to change it after creation) with non-zero elements of items. Use filter() method for this. 

const cleaned = items.filter(...);
How to filter non-zero elements?

Since 0 has logical value false and all other numbers have true, you may filter non-zero items by their logical value. To get boolean value of something, use Boolean() function. If you use this function as filtering rule, it keeps only true elements. 

const cleaned = items.filter(Boolean);
What to do next?

Now you need to sort cleaned descending, to later get elements from it in right order - from the end from smallest. Use sort() method for this with anonymous function for sorting descending. 

const cleaned = items.filter(Boolean).sort((a, b)=> b-a);

How to fill the new array?

Iterate over the items with for/of loop and push the result of an expression to the new array. 

for (let i of items)
    res.push(...);
Help me to write expression.

Let's think together, what should be there. If the value of current item of items is 0 - it should remain its place in the final array, otherwise - next value from the sorted array should be paste in the final array. This way you fill the final array with 0 or values in ascending order. It looks like you need ternary operator here with the checking the logical value of item, as we did before. 

Boolean(i) ? ... : ...
How to finish the expression?

When current element is non-zero (boolean value == true), you must put the next value from your sorted array cleaned. In this case, a short and easy way to do it is using pop() method - it removes the last element and return it. Since cleaned is sorted descending you always need the last (smallest) element. 0 remains 0. 

Boolean(i) ? cleaned.pop() : 0
So, how my final solution should look like?

You just need to join everything and wrap your code into function expression.

Here is the whole solution:

function exceptZero(items: number[]): number[] {

    const cleaned = items.filter(Boolean).sort((a, b)=> b-a);
    let res = [];
    for (let i of items)
        res.push(Boolean(i) ? cleaned.pop() : 0);
    return res;
}
